# 2.12 ScrollPane
## 1) ScrollPane
- 다른 컴포넌트를 포함시킬 수 있는 컨테이너
- 다른 컨테이너들과 달리, 단 하나의 컴포넌트만을 포함시킬 수 있다.
- 제한된 공간에서 크기가 큰 컴포넌트를 화면에 보여줄 수 있도록 하는데 사용
- 포함된 컴포넌트의 크기가, Scrollbar 자신보다 클 경우 스크롤바를 이용해서 볼 수 있게 함.
