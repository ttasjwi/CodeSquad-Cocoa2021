# 2.13 Dialog
## 1) Dialog
- 화면에 메시지창을 보여주는데 사용.
- 프로그램 실행 중, 에러가 발생했음을 알리거나, 파일을 사용하기 전, 사용자로부터 응답을 받아야한다거나 하는 상황 등에 사용.
- 다른 컴포넌트를 포함시킬 수 있는 컨테이너. Frame과 유사한 모양을 하고 있음.
