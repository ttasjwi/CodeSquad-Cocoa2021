# 2.7 TextField
## 1) TextField
- 사용자로부터 값을 입력받을 수 있는 컴포넌트.
- 수정이 가능하다.
- 한 줄(라인)만 입력받음.
- id, password, 이름 등 비교적 짧은 값의 입력에 사용됨
